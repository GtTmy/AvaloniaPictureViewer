# AvaloniaPictureViewer

![gui-image](readme-image/gui-image.png)

## 概要
- Avaloniaのデモ目的です
- カレントディレクトリに配置された`image`フォルダ内のjpgファイルを順番に表示します
- ウインドウタイトルに、アプリを起動しているOSの名前を表示します

## 開発環境
- Windows10 Home 1803
- Visual Studio 2019 Community Edition
